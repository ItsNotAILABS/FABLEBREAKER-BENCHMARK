Associate Editors

Each journal track is overseen by an associate editor with domain expertise.

Journal of Adversarial Evaluation

Focus: Adversarial test generation, evasion strategies, evaluator stress testing

Seeks papers that advance the theory and practice of generating test cases that expose correctness failures in optimized systems. Contributions must demonstrate novel failure modes or improved detection rates over existing approaches.

Journal of Benchmark Architecture

Focus: Evaluation system design, scoring protocols, gaming resistance

Seeks papers on the formal design of evaluation infrastructure that resists manipulation. Contributions should address structural properties of benchmark systems rather than individual test cases.

Journal of Certification Systems

Focus: Cryptographic evidence, trust protocols, governance models

Seeks papers that formalize the trust assumptions underlying benchmark certification and propose mechanisms to strengthen evidence integrity. Contributions must address both technical and procedural aspects of certification.

Journal of Semantic Preservation

Focus: Formal verification, evaluator equivalence, correctness proofs

Seeks papers that advance formal methods for verifying that optimized evaluators preserve semantic equivalence with reference implementations. Contributions should provide mathematical rigor beyond empirical testing alone.

Journal of Reproducibility Methods

Focus: Deterministic generation, measurement frameworks, automation

Seeks papers that improve the reproducibility of benchmark results through better generation algorithms, statistical methods, or automation infrastructure. Contributions should demonstrate measurable reduction in measurement variance or process variability.

Review Standards

All submissions undergo rigorous peer review against the following criteria.

Technical Correctness

Claims must be supported by formal argument, empirical evidence, or both. Mathematical statements must be provably correct. Experimental results must be reproducible from described methodology.

Novelty

Submissions must present ideas, methods, or results not previously published in the FableBreaker research corpus. Incremental improvements over prior work are acceptable if the improvement is clearly quantified and significant.

Relevance

Papers must directly advance the FableBreaker project's mission of rigorous, adversarial, reproducible benchmark certification. Pure theoretical contributions are welcome if they have clear implications for benchmark practice.

Clarity

Papers must be written clearly enough that a competent practitioner in the field can understand and implement the described methods. Notation must be defined, assumptions stated, and limitations acknowledged.

Submission Guidelines

Format

All papers are published in HTML format using the FableBreaker paper template. Authors should use semantic HTML with the provided CSS classes. Papers should include: title, authors, affiliation, abstract, keywords, numbered sections, and references.

Process

Submit papers via pull request to the journal/ directory. Include the paper HTML file in the appropriate journal subdirectory. Update the journal index.html to include the new paper in the paper list. Allow 14 days for editorial review.

Open Access

All FableBreaker research is published under open access. There are no submission fees, page charges, or access restrictions. The research is funded by ItsNotAI LABS and made freely available to advance the field of benchmark integrity.
